# Sahudyscos

## Dependencies

- Java 8+ (`sudo apt install openjdk-8-jre`)
- Maven (`sudo apt install maven`)
- Tomcat 7+ (`sudo apt install tomcat7`)

## Building

`mvn package`

## Running

`mvn spring-boot:run`

**OR**

Copy the WAR file inside `target` into the `webapps` Tomcat folder
